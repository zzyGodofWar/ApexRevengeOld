# ApexRevengeOld
APEX：英雄   
写成了可商用游戏外挂（现已停止更新并已被检测，仅作为学习交流使用）  

独立开发，此为Ring3 应用层源码，驱动源码暂不开源

登录界面：Dx9自绘  
功能菜单：基于Imgui dx9绘图  
基本功能：透视、自瞄、压枪、显示全局游戏信息
